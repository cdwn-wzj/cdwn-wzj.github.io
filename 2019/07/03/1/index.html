前言

1. 使用技术

  • Hexo 是一款基于 Node.js 的静态博客框架。Hexo 使用 Markdown 解析文章，用户在本地安装Hexo并进行写作，通过一条命令，Hexo即可 利用靓丽的主题自动生成静态网页。
  • GitHub Pages 存储库

2.使用github pages服务搭建博客的好处

  1. 全是静态文件，访问速度快；
  2. 免费方便，不用花一分钱就可以搭建一个自由的个人博客，不需要服务器不需要后台；
  3. 可以随意绑定自己的域名，不仔细看的话根本看不出来你的网站是基于github的；
  4. 数据绝对安全，基于github的版本管理，想恢复到哪个历史版本都行；
  5. 博客内容可以轻松打包、转移、发布到其它平台；
  6. 等等；

相关步骤

1. 安装Git

  • Git官方下载地址下载完成后傻瓜式安装       建议:使用网络加速器下载
  • 安装完成后到桌面单击右键出现如下图所示的两个git就说明成功了
  • 然后使用Git Bash Here修改一下自己的email和name（以防止后面报错）
  • 使用命令查看是否修改成功

2. 安装Node.js

  • Node.js官方下载地址下载完成后傻瓜式安装
  • 安装完成后会出现如下图的这两个图标

3. 安装cnpm

  • 启动命令行
  • 输入:npm install -g cnpm –registry=https://registry.npm.taobao.org

4. 安装hexo

  • 在命令行输入cnpm install -g hexo-cli

5. 创建blog目录

  • 随便找个位置创建博客目录
  • 如我创建的blog目录就在(E:\myblog)

6. 初始化博客

  • 使用命令行进入博客目录
  • 在博客目录下输入:hexo init

7. 启动本地博客

  • 在博客目录下输入:hexo server（或hexo s）
  • 显示如下图这就说明hexo启动成功并且端口为4000
  • 然后我们在浏览器上输入:localhost:4000

8. 在GitHub上创建仓库

  • 登录GitHub（没有的自己建个账号）
  • 创建New repository(新存储库)
  • Repository name输入：自己用户名.github.io然后 Create repository

  • 在文件目录下输入：hexo d
